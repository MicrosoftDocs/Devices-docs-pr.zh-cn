# <a name="posture"></a>状态

**某人可以保留其正文的位置。**

状态和持久性将一起提供。 我们能否将我们的实体维护到某个位置可以确定我们如何与环境交互。 如果某人在驾驶汽车时无法保持直立的状况，则很难安全地看到防风和驾驶。 使用技术所必须维持的状况同样要求很高。 如果某人无法保持直立的形态，他们可能很难坐在桌面上。

对于具有如证书的易发性等条件的人，使用登录选项（如面部识别）而不是键入可能更容易。 但是，如果用户无法保持头部和直立状态，他们可能会发现很难使用面部识别。

状态可能会受影响声调、强度、对一个人正文的控制、传感器输入、属性感知输入或使维护特定位置变得困难的任何条件的影响。 这可能包括 Parkinson、amyotrophic 横向 (ALS) 、) 、笔划、多发等。


## <a name="barriers"></a>障碍
* 设备使用 (的正文定位需求，例如，坐在一个立着的座位上，用双手从桌面上提取来使用台式计算机，或者用两只手按住控制器，以成功玩游戏) 

## <a name="facilitators"></a>推动者
* 灵活的输入方法选项，允许用户在自己选择的位置访问设备 (例如，可位于用户单圈上的 蓝牙 键盘，而不是桌面) 
* 便于用户在输入模式期间选择的辅助 (例如，动物鼠标、自适应技术按钮和开关支持) 
* 提供和支持的装载材料，允许用户在自己选择的位置访问设备

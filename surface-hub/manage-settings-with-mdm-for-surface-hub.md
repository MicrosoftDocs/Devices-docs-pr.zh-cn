# <span data-ttu-id="8b054-104">使用 MDM 提供程序管理设置 (Surface Hub)</span><span class="sxs-lookup"><span data-stu-id="8b054-104">Manage settings with an MDM provider (Surface Hub)</span></span>

<span data-ttu-id="8b054-105">Surface Hub 和其他 Windows10 设备支持 IT 管理员使用移动设备管理 (MDM) 提供程序管理设置和策略。</span><span class="sxs-lookup"><span data-stu-id="8b054-105">Surface Hub and other Windows 10 devices allow IT administrators to manage settings and policies using a mobile device management (MDM) provider.</span></span> <span data-ttu-id="8b054-106">内置管理组件与管理服务器通信，因此无需在设备上安装其他客户端。</span><span class="sxs-lookup"><span data-stu-id="8b054-106">A built-in management component communicates with the management server, so there is no need to install additional clients on the device.</span></span> <span data-ttu-id="8b054-107">有关详细信息，请参阅 [Windows10 移动设备管理](https://msdn.microsoft.com/library/windows/hardware/dn914769.aspx)。</span><span class="sxs-lookup"><span data-stu-id="8b054-107">For more information, see [Windows 10 mobile device management](https://msdn.microsoft.com/library/windows/hardware/dn914769.aspx).</span></span>

<span data-ttu-id="8b054-108">Surface Hub 已通过 Microsoft 的第一方 MDM 提供程序进行验证：</span><span class="sxs-lookup"><span data-stu-id="8b054-108">Surface Hub has been validated with Microsoft's first-party MDM providers:</span></span>

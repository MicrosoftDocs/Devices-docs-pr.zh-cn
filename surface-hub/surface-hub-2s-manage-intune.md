# <span data-ttu-id="24524-104">使用 Intune 管理 Surface Hub 2S</span><span class="sxs-lookup"><span data-stu-id="24524-104">Manage Surface Hub 2S with Intune</span></span>

## <span data-ttu-id="24524-105">使用 Intune 注册 Surface Hub 2S</span><span class="sxs-lookup"><span data-stu-id="24524-105">Register Surface Hub 2S with Intune</span></span>

<span data-ttu-id="24524-106">Surface Hub 2S 支持 IT 管理员使用移动设备管理 (MDM) 提供程序管理设置和策略。</span><span class="sxs-lookup"><span data-stu-id="24524-106">Surface Hub 2S allows IT administrators to manage settings and policies using a mobile device management (MDM) provider.</span></span> <span data-ttu-id="24524-107">Surface Hub 2S 具有内置管理组件，可与管理服务器通信，因此无需在设备上安装其他客户端。</span><span class="sxs-lookup"><span data-stu-id="24524-107">Surface Hub 2S has a built-in management component to communicate with the management server, so there is no need to install additional clients on the device.</span></span>

### <span data-ttu-id="24524-108">手动注册</span><span class="sxs-lookup"><span data-stu-id="24524-108">Manual registration</span></span>

1. <span data-ttu-id="24524-109">打开 Surface Hub 2 的 " **设置** " 应用，并以本地管理员身份登录。</span><span class="sxs-lookup"><span data-stu-id="24524-109">Open the **Settings** app on the Surface Hub 2S and sign in as a local administrator.</span></span> <span data-ttu-id="24524-110">选择“**Surface Hub**” > “**设备管理**”，然后选择 **+** 进行添加。</span><span class="sxs-lookup"><span data-stu-id="24524-110">Select **Surface Hub** > **Device management** and then select **+** to add.</span></span>
2. <span data-ttu-id="24524-111">系统将提示你使用要用于 Intune 的帐户登录。</span><span class="sxs-lookup"><span data-stu-id="24524-111">You will be prompted to login with the account to use for Intune.</span></span> <span data-ttu-id="24524-112">进行身份验证后，该设备将自动使用 Intune 进行注册。</span><span class="sxs-lookup"><span data-stu-id="24524-112">After authenticating, the device will automatically register with Intune.</span></span>

   ![使用 Intune 注册 Surface Hub 2S](images/sh2-set-intune1.png)<br>
   
> [!NOTE]
> <span data-ttu-id="24524-114">用于身份验证的帐户将是 Intune 注册帐户，并且必须获得 Intune 许可。</span><span class="sxs-lookup"><span data-stu-id="24524-114">The account used for authentication will be the Intune enrollment account, and must be licensed for Intune.</span></span>

### <span data-ttu-id="24524-115">自动注册 — 与 Azure Active Directory 关联</span><span class="sxs-lookup"><span data-stu-id="24524-115">Auto registration — Azure Active Directory Affiliated</span></span>

<span data-ttu-id="24524-116">在初始设置流程中，将 Surface Hub 与已启用 Intune 自动注册的 Azure AD 租户相关联时，设备将自动使用 Intune 注册。</span><span class="sxs-lookup"><span data-stu-id="24524-116">During the initial setup process, when affiliating a Surface Hub with an Azure AD tenant that has Intune auto enrollment enabled, the device will automatically enroll with Intune.</span></span> <span data-ttu-id="24524-117">有关详细信息，请参阅[适用于 Windows 设备的 Intune 注册方法](https://docs.microsoft.com/intune/enrollment/windows-enrollment-methods)。</span><span class="sxs-lookup"><span data-stu-id="24524-117">For more information, refer to [Intune enrollment methods for Windows devices](https://docs.microsoft.com/intune/enrollment/windows-enrollment-methods).</span></span> <span data-ttu-id="24524-118">Surface Hub 需要进行 Azure AD 关联和 Intune 自动注册才能成为 Intune 中的“合规设备”。</span><span class="sxs-lookup"><span data-stu-id="24524-118">Azure AD affiliation and Intune auto enrollment is required for the Surface Hub to be a "compliant device" in Intune.</span></span>

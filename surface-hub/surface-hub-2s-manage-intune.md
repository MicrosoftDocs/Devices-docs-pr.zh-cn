# <span data-ttu-id="90289-104">使用 Intune 管理 Surface Hub 2S</span><span class="sxs-lookup"><span data-stu-id="90289-104">Manage Surface Hub 2S with Intune</span></span>

## <span data-ttu-id="90289-105">使用 Intune 注册 Surface Hub 2S</span><span class="sxs-lookup"><span data-stu-id="90289-105">Register Surface Hub 2S with Intune</span></span>

<span data-ttu-id="90289-106">Surface Hub 2S 支持 IT 管理员使用移动设备管理 (MDM) 提供程序管理设置和策略。</span><span class="sxs-lookup"><span data-stu-id="90289-106">Surface Hub 2S allows IT administrators to manage settings and policies using a mobile device management (MDM) provider.</span></span> <span data-ttu-id="90289-107">Surface Hub 2S 具有内置管理组件，可与管理服务器通信，因此无需在设备上安装其他客户端。</span><span class="sxs-lookup"><span data-stu-id="90289-107">Surface Hub 2S has a built-in management component to communicate with the management server, so there is no need to install additional clients on the device.</span></span>

### <span data-ttu-id="90289-108">手动注册</span><span class="sxs-lookup"><span data-stu-id="90289-108">Manual registration</span></span>

1. <span data-ttu-id="90289-109">以本地管理员身份登录 Surface Hub 2S，然后打开“**设置**”应用。</span><span class="sxs-lookup"><span data-stu-id="90289-109">Sign in as a local administrator on Surface Hub 2S and open the **Settings** app.</span></span> <span data-ttu-id="90289-110">选择“**Surface Hub**” > “**设备管理**”，然后选择 **+** 进行添加。</span><span class="sxs-lookup"><span data-stu-id="90289-110">Select **Surface Hub** > **Device management** and then select **+** to add.</span></span>
2. <span data-ttu-id="90289-111">进行身份验证后，该设备将自动使用 Intune 进行注册。</span><span class="sxs-lookup"><span data-stu-id="90289-111">After authenticating, the device will automatically register with Intune.</span></span>

   ![使用 Intune 注册 Surface Hub 2S](images/sh2-set-intune1.png)<br>

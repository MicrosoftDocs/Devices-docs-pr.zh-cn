# <a name="install-and-mount-surface-hub-2s-85"></a>安装和装载 Surface Hub 2S 85"

本文提供有关在商业环境中实际安装 Microsoft Surface Hub 2S 85"的支持指南。 

## <a name="unboxing-video"></a>取消对视频进行装箱

现已提供用于移动和安装 Surface Hub 2S 85 的配套视频。 开始之前，请查看 Microsoft [Surface Hub 2S 85" 取消装箱和设置](https://www.microsoft.com/zh-cn/videoplayer/embed/RE4MRqV)。  

## <a name="follow-all-safety-precautions"></a>遵循所有安全预防措施 

> [!WARNING]
> **处理和网站准备**<br>
> - 设备很重。 若要降低因设备大小和权重导致个人受损、死或受损的风险，使设备保持直立很重要。
>- 在将设备移动到安装设备的位置之前，请调查网站以确定如何安全地将其移动到将解压缩和装入的位置。
>- 始终使用至少两个人进行解压缩和安装。
>- 设备解压缩后，应立即装入，以便安装系统在解压缩之前就位。 如果要装载到滚动的立架上，在解压缩之前锁定或阻止该架的方向盘。
>- 为了避免脱行危险，请使装配区域清除打包材料。

> [!IMPORTANT]
>在继续之前，请查看以下附录中列出的其他 [安全](#appendix) 信息。

## <a name="1-remove-outer-packaging-materials"></a>1. 删除外部打包材料

1. 删除并回收外部覆盖。
2. 剪切四个 (4 个) 条。
3. 从两端打开并删除六 (6) 剪辑。
4. 移除顶盒，然后抬起并移除前后面板。

 ![删除外部打包](images/hub85mount1a.png) <br>

## <a name="2-remove-black-outer-packaging-foam"></a>2. 删除黑色外部打包包装

1. 删除 4 号 (角) 。
2. 删除黑色中心填充支持。

 ![删除黑色外部打包包装](images/hub85mount2a.png) <br>

> [!Caution]
> 在 Hub 2S 与要放置它的购物车或墙壁装载相邻之前，不要删除任何白色包装、固定或打包材料。
其他固定材料在小框中提供，该小框位于包背面的手风箱下。 在移动之前（尤其是在旋转设备及其保护包之前）必须安装原始或替换的固定材料
 

## <a name="3-remove-inner-packaging-frame-from-pallet"></a>3. 从托盘中删除内部打包框架

1. 将托盘程序集移动到包装位置。
2. 锁轮将 (4) 。
3. 从托盘上滑动内部打包。

 ![从托盘中删除内部打包框架](images/hub85mount3b.png) <br>

## <a name="4-rotate-packaging-frame-to-fit-elevator"></a>4. 旋转打包框架以适应包装

1. 旋转框架以适应提升
2. 在内部框架的斜面端旋转，然后滑入门中。

 ![旋转打包框架以适应包装](images/hub85mount4a.png) <br>

> [!TIP]
> 轮式托盘可自定义适合内部框架打包占用，并且可在安装网站交付过程中使用。 内部木框架端部分具有一些防滑印板。
 
## <a name="5-remove-from-elevator"></a>5. 从中删除

1. 滑出幻灯片
2. 锁轮 (x4) 。

 ![从安装中删除](images/hub85mount5a.png) <br>


## <a name="6-place-surface-hub-85-back-on-pallet"></a>6. 将 Surface Hub 85" 重新放在托盘上

1. 使用斜面端，旋转回托盘。
2. 解锁方向盘 (x4) 。
3. 将产品移动到墙壁或购物车装载的位置。

 ![将 Surface Hub 85" 重新放在托盘上](images/hub85mount6a.png) <br>

## <a name="7-place-surface-hub-85-on-wall-mount-or-cart"></a>7. 将 Surface Hub 85" 放在挂载或购物车上

1. 将 Hub 2S 放在墙壁装载或购物车的前面。
2. 锁轮 (x4) 。
3. 从托盘上滑动内部打包。
4. 剪切 3 个密封带。
5. 删除顶盒。
6. 删除白色填充部分。
7. 删除欢迎工具包。
8. 通过垂直抬起来删除结束部分。
9. 通过四手装饰在木底上清除木端部分。

   ![将 Surface Hub 85" 放在挂载或购物车上](images/hub85mount7a.png) <br>

   > [!CAUTION]
   >  不要将 Hub 2S 85" 保持无人参与状态。 需要一个其他人来使设备保持垂直。 删除结束部分后，至少需要一个人与 Hub 2S 保持接触，直到放置到墙壁装载或购物车中。

10. 向上和后退抬起百叶木末端。
11. 从后面删除凹凸标签。
12. 从下纸盒抬起 Hub 2S，放在购物车或墙壁装载上。

    > [!NOTE]
    > 按照购物车或墙壁装载制造商说明，在将 Hub 2S 从内部打包纸盒中删除之前准备安装系统。
 
    > [!TIP]
    > 在掌握 Hub 2S 时，请注意手部在下半角大写中切开。 必须小心使用顶手，不要抓取扬声器所在的设备。 设备封面两端的图形提供一般扬声器位置。

13. 将弹性固定剪辑 (x2) 。
14. 删除覆盖层。
15. 记下将笔 (x2) 、相机和电缆的位置。
16. 将笔 (x2) 、相机和电缆。
17. 删除 x4 (标签) 。
18. 按右下角的电源按钮。 安装现已完成。

## <a name="appendix"></a>附录 

### <a name="additional-safety-information"></a>其他安全信息

> [!WARNING]
> **重对象/结构提升**<br>
> 设备很重。 为了降低设备因抬起而受损、死或受损的风险，我们建议至少两名或两名以上人员抬起设备。 在抬起和/或移动设备时，使用正确的抬起状态非常重要。
> 使用良好的结构提升做法，包括但不限于：
> - 提前规划。 确保提升团队同意该计划。
>- 确定是否可以抬起设备。 它太重还是太难看？
>- 确定是否需要抬起辅助设备。
>- 检查您的环境是否出现障碍物和滑滑表面。
>- 抬起你的两只脚，而不是背。
>- 弯曲一下，保持后面直直。
>- 使单位靠近正文。
>- 将正文居中位于单位上。 将双脚与双边的宽度分开。
>- 平稳地向上直抬起。
>- 保持直视;提升时或提升负载后不要扭曲


> [!WARNING]
>  **正确安装**<br> 设备很重，并连接到购物车或墙壁装载。 若要降低设备受损、死或受损的风险，请执行以下操作： 
>- 按照购物车或挂载制造商提供的所有说明进行操作。
>- 确保建议的安装系统将支持此设备的权重。
>- 仅使用随系统安装一起提供的安装硬件。
>- 确保所有产品都按照制造商的说明进行安全强化。
>- 在确保设备完全使用安装系统附件点之前，不要释放设备。
>- Microsoft 建议使用设计为与设备一同使用的购物车或墙壁安装系统。 Microsoft 对因使用其他安装系统导致的任何损害、损害或死负责。

> [!WARNING]
>  **墙壁或其他安装表面未看见的危害** <br>
墙壁和其他安装表面可能包含电力线、管道和其他无法看见的障碍物。 剪切或钻取无法看见的危害可能会导致严重的个人损害或死。
安装程序负责在安装之前查找未检测到的潜在危险，并避免这些危险。 评估安装环境，并始终确保在钻取和/或剪切之前，墙壁或其他安装表面没有未看见的危害。

> [!WARNING]
> **提示危险**<br>
> 为避免在移动购物车/独立安装设备时造成个人损失、死或损坏的风险：
> - 只能使用与此设备兼容的购物车/架。
>- 按照购物车/独立制造商提供的所有说明移动或重新定位独立安装的设备。
>-  请勿在设备或购物车/架上挂起或放置大量对象。
>-  在移动购物车/独立安装的设备之前，根据需要断开电缆和其他电缆。 在移动购物车/独立安装设备时，请谨慎操作并缓慢移动。 按照购物车/独立制造商的说明移动或重定位该架。
>-  在将购物车/独立安装的设备上斜或下斜传输时，请谨慎。 切勿将购物车/独立装载的设备无人值守在渐变上或附近。
>-  仅成人应移动购物车/独立安装的设备。

> [!CAUTION]
> **触摸屏镜**<br>
与大多数触摸屏一样，设备的触摸屏由镜面组成。 如果设备被丢弃或收到显著影响，则此镜可能会中断。 若要降低个人损害的风险，请避免触摸屏幕（如果镜被打破、切开或开开，并安排更换屏幕）。
产品有限担保中未涵盖因滥用或滥用设备而引发的损坏或缩小的触摸屏。

> [!WARNING]
>  **正确安装**<br>
为了避免与不正确的设备安装相关的危险，安装必须由在开始工作之前已阅读并了解安装说明的人执行。 如果你没有必要的设备或专业知识，或者不确定安装表面能否正确支持，请咨询专业安装程序。

# 安装和装载 Surface Hub 2S 85"

本文提供在商业环境中以物理方式安装 Microsoft Surface Hub 2S 85" 的支持指南。 

## 取消对视频进行装箱

现已提供用于安装 Surface Hub 2S 85 的配套视频。 开始之前，请查看取消 [装箱和安装 Surface Hub 2S 85"](https://aka.ms/Hub2S85Unboxing)。

## 遵循所有安全预防措施 

> [!WARNING]
> **处理和网站准备**<br>
>- 设备非常重。 若要降低因设备大小和权重而造成个人受损、死或损坏的风险，使设备保持直立很重要。
>- 在将设备移动到安装设备的位置之前，请调查网站以确定如何安全地将其移动到将解压缩和装入的位置。
>- 始终使用至少两个人进行解压缩和安装。
>- 设备解压缩后，应立即装入，以便安装系统在解压缩之前应已就位。 如果你要装载到滚动的台子上，在解压缩之前锁定或阻止该台的方向盘。
>- 为了避免出现意外危险，请保持程序集区域清除打包材料。

> [!IMPORTANT]
>在继续之前，请查看以下附录中列出的其他 [安全](#appendix) 信息。

## 1. 删除外部打包材料

1. 删除并回收外部覆盖。
2. 将四个 (4) 固定在一起。
3. 从两端打开并 (6) 6 个剪辑。
4. 移除顶板，然后抬起并移除前后面板。

 ![删除外部打包](images/hub85mount1a.png) <br>

## 2. 删除黑色外部打包容器

1. 从 4 (中删除黑色角) 。
2. 删除黑色中心填充支持。

 ![删除黑色外部打包包装](images/hub85mount2a.png) <br>

> [!Caution]
> 在 Hub 2S 与要放置它的购物车或墙壁挂载相邻之前，不要删除任何白色容器、固定线或包装材料。
其他固定材料在小框中提供，该小框位于包背面的手风箱下。 在移动之前，尤其是旋转设备及其保护包之前，原始或替换的固定材料必须就位
 

## 3. 从托盘中删除内部打包框架

1. 将托盘程序集移动到位置。
2. 锁轮方向盘 (4) 。
3. 从包装板上滑动内部打包。

 ![从托盘中删除内部打包框架](images/hub85mount3b.png) <br>

## 4. 旋转打包框架以适应容器

1. 旋转框架以适应边框
2. 在内部框架的斜面末端旋转，然后滑动到筒中。

 ![旋转打包框架以适应容器](images/hub85mount4a.png) <br>

> [!NOTE]
> 滚轮式托盘可自定义适合内部框架打包占用，并可在安装网站交付过程中使用。 内部框架端部有一些防滑板。
 
## 5. 从安装中删除

1. 滑出
2. 锁轮方向盘 (x4) 。

 ![从中删除](images/hub85mount5a.png) <br>


## 6. 将 Surface Hub 85" 重新放在托盘上

1. 使用斜面，旋转回托盘。
2. 解锁方向盘 (x4) 。
3. 将产品移动到墙壁或购物车装载的位置。

 ![将 Surface Hub 85" 重新放在托盘上](images/hub85mount6a.png) <br>

## 7. 将 Surface Hub 85"放在挂载或购物车上

1. 将 Hub 2S 放在墙壁装载或购物车的前面。
2. 锁轮方向盘 (x4) 。
3. 从包装板上滑动内部打包。
4. 剪切 3 个密封带。
5. 删除外盖。
6. 删除白色花条。
7. 删除欢迎工具包。
8. 通过垂直抬起删除结束部分。
9. 通过四个手形来移除木底上的木端部分。

   ![将 Surface Hub 85"放在挂载或购物车上](images/hub85mount7a.png) <br>

   > [!CAUTION]
   >  不要使 Hub 2S 85" 无人值守。 需要其他人员来使设备保持垂直。 删除结束部分后，至少需要一个人与 Hub 2S 保持联系，直到安装在墙壁或购物车上完成。

10. 向上和后退抬起木纸末端部分。
11. 从后面删除凹凸标签。
12. 将 Hub 2S 从下层纸盒中抬起，并放在购物车或挂墙上。

    > [!NOTE]
    > 按照购物车或挂载制造商说明进行操作，在将 Hub 2S 从内部打包纸盒中删除之前准备安装系统。
 
    > [!NOTE]
    > 当掌握 Hub 2S 时，请注意手放在下半角的切口。 必须小心使用上手，不要掌握扬声器所在的设备。 设备封面两端的图形提供一般扬声器位置。

13. 将弹性剪辑 (x2) 。
14. 删除覆盖层。
15. 记下将笔 (x2) 相机和电缆的位置。
16. 将笔 (x2) 、相机和电缆。
17. 删除 x4 (标签) 。
18. 按右下角的电源按钮。 安装现已完成。

### 其他安全信息

> [!WARNING]
> **重对象/任务提升**<br>
> 设备非常重。 为了降低与抬起相关的损失、设备死或损坏的风险，我们建议至少两名或两名以上人员抬起设备。 在抬起和/或移动设备时，使用适当的抬起状态非常重要。
> 使用良好的步法提升做法，包括但不限于：
> - 提前规划。 确保提升团队同意该计划。
>- 确定是否可以抬起设备。 它太重还是太难看？
>- 确定是否需要抬起辅助设备。
>- 检查你的环境是否出现障碍物和障碍物表面。
>- 抬起你的两只脚，而不是背。
>- 在台式中弯曲，保持背面直。
>- 使单位靠近正文。
>- 将正文居中位于单位上。 将双脚与双头宽度分开。
>- 平滑向上抬起。
>- 保持直视;在抬起时或抬起负载后不要旋转


> [!WARNING]
>  **正确安装**<br> 设备很重，并连接到购物车或挂墙。 若要降低设备遭受损害、死或损坏的风险： 
>- 按照购物车或挂载制造商提供的所有说明进行操作。
>- 确保建议的装载系统将支持此设备的权重。
>- 仅使用随系统安装一起提供的安装硬件。
>- 确保根据制造商的说明安全强化所有产品。
>- 在确保设备完全使用装载系统附件点之前，不要释放设备。
>- Microsoft 建议使用设计为与设备一同使用的购物车或墙壁装载系统。 Microsoft 对因使用其他装载系统引起的任何损害、损害或死负责。

> [!WARNING]
>  **墙壁或其他装载表面中未看见的危险** <br>
墙壁和其他装载表面可能包含电力线、管道和其他未看到的危险或障碍物。 剪切或钻取无法看见的危险可能导致严重的个人损害或死。
在安装过程中，安装程序负责查找之前未看见的危险，并避免这些危险。 评估装载环境，并始终确保在钻取和/或剪切之前，墙壁或其他安装表面没有未看见的危险。

> [!WARNING]
> **提示危险**<br>
> 为了避免在移动购物车/独立安装设备时造成个人损失、死或损坏的风险：
> - 只能使用与此设备兼容的购物车/购物车。
>- 按照购物车/独立制造商提供的所有说明移动或重新定位独立安装的设备。
>-  请勿从设备或购物车/购物车/购物车中挂起或放置大量对象。
>-  在移动购物车/独立安装的设备之前，根据需要断开电缆和其他电缆。 移动购物车/独立安装的设备时，请谨慎操作并缓慢移动。 按照购物车/独立制造商的说明移动或重新定位该部件。
>-  在将购物车/独立装载的设备向上或向下渐变传输时，请谨慎。 切勿将购物车/独立安装的设备无人值守在渐变上或附近。
>-  只有小马才能移动购物车/独立装载的设备。

> [!CAUTION]
> **触摸屏镜**<br>
与大多数触摸屏一样，设备的触摸屏由镜面组成。 如果设备被丢弃或受到显著影响，则此镜可能会中断。 若要降低个人崩溃的风险，请避免触摸屏幕（如果损坏、切掉或破解了镜面并安排更换屏幕）。
由滥用或滥用设备导致的损坏或缩小的触摸屏不在产品的有限担保范围内。

> [!WARNING]
>  **正确安装**<br>
为了避免与不正确的设备安装相关的危险，安装必须由在开始工作之前阅读并了解安装说明的人执行。 如果你没有必要的设备或专业知识，或者不确定装载表面能否正确支持，请咨询专业安装程序。
